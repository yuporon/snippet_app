# スニペットアプリ

## 概要

### リンク
[https://memoporon.herokuapp.com/](https://memoporon.herokuapp.com/)

### 目的
- Axiosを使ってみる
- Vue.jsのUIライブラリであるVuetifyを学ぶ
- HEROKUでデプロイ

### 内容
- メモの追加
- メモの削除
- メモの更新
- メモをマークダウン表示
- 検索機能

### 環境
- MacOS BigSur 11.2.3
- Ruby 2.5.1
- Ruby on Rails 5.2.3
- SQLite 3.28.0
- Vue.js 2.6.10
- Vuetify 1.5
